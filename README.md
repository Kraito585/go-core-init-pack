# go-core-by-kraito

## Getting started

Требуется установленый make на устройстве с которого идёт разработка linux/mac имеют его по умолчанию.



Init pack для разработки микросервисов под k8s.

Умеет работать с кластерами redis и postgresql, S3 сервис протестирован на RustFS и S3 Selectel, готовые решения для Kafka, Cors в двух вариациях фиксированый pool из конфига для не требывательных задач и хранения Cors запей в redis, middleware инструменты для работы с JWT, rate limit.

Возможно что то ещё, пока у меня нет нужды потдерживать я его лично не буду если комуто понравится можете контрибютить измененя я их изучу и добавлю.
